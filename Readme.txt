이 프로그램은 소프트웨어 설계 시간에 작성한 교환 학생 프로그램을 수정, 발전하여 만든 프로그램으로
김근우, 김영훈, 이지수, 한제규, 황민태 5명이 개발하였습니다.

1. 로그인에 관하여
현재 DB에 저장되어있는 사용자는 학생 2명, 관리자 1명으로 아이디와 비밀번호는 다음과 같습니다.

	ID	PW
학생	s1123	1123
학생	s1132	1132
관리자	a1124	1124

이전과 달리 등록되있지 않은 사용자도 새로 추가된 회원가입 기능을 이용해서 새로운 계정을 만들어서 프로그램에 접속할 수 있게 되었습니다. (단 학생 계정만 가능합니다)
따라서 비회원 로그인 기능이 불필요하다 생각하여 기능을 제거하였습니다.

로그인 후 사용할 수 있는 기능은 이전과 같이 각 계정별로 실행할 수 있는 권한이 다르며 이는 아래에서 설명하겠습니다.

2. 각 기능에 대하여
1) 회원가입 기능
- 새로 추가된 기능으로 등록되지 않은 사용자도 회원가입창을 이용하여 새로운 계정을 만들 수 있습니다. 관리자 계정은 따로 만들 수 없으며 학생 계정만 만들 수 있습니다. 다른 부분은 반드시 입력되어야 하지만 이메일과 전화번호는 없을 수도 있을 것 같아서 빈 칸으로 입력가능합니다.

2) 모집공고 조회
- 모든 사용자가 사용가능한 기능이며 모집공고가 존재하면 중앙의 리스트에 모집공고 목록이 나타납니다. 이 리스트에서 보고싶은 모집 공고를 더블 클릭하면 새 창으로 그 모집공에 대한 세부 정보를 볼 수 있으며 학생의 경우 지원하기 버튼을 눌러 그 모집공고에 지원할 수 있습니다.

3) 진행상황 조회
- 학생만 사용가능한 기능이며 학생 별로 다른 결과가 나타납니다. 현재 접속한 학생이 어떤 모집공고에 대해 응시원서를 작성했다면 그에 대한 진행상황을 조회하고 최종합격 등록 및 삭제를 할 수 있습니다.

4) 이수학점 관리
- 학생만 사용가능한 기능이며 학생 별로 다른 결과가 나타납니다. 현재 접속한 학생이 이전에 갔다온 교환학생 프로그램에 대한 이수학점이 존재하면 그 이수학점에 대해 등록신청을 할 수 있고 그 학점이 제대로 신청되었는지 확인할 수 있습니다.

5) 파견실적 조회
- 이전에 구현하지 못했던 기능을 이번에 새로 구현하였습니다. 국가, 대학, 전공, 기간을 기준으로 검색할 수 있으며 선택하지 않은 부분에 대해서는 전체를 검색합니다. 즉, 국가를 미국으로 설정하고 대학을 미선택, 전공을 컴퓨터전공, 기간을 미선택하면 미국의 컴퓨터 전공에 대한 파견실적을 전체 검색합니다. 따라서 전부 미선택으로 할 경우 최근 6년 동안의 전체 기록을 확인할 수 있습니다. 그 내용은 그래프를 그려서 나타냅니다.

6) 질문게시판
- 모든 사용자가 사용 가능한 기능이며 세부적으로는 QnA 게시판과 FAQ 게시판이 있습니다. 학생은 QnA게시판에 질문을 등록할 수 있으며 관리자는 FAQ를 등록하고 학생이 올린 질문에 답변할 수 있습니다.

7) 모집공고 작성
- 관리자만 사용가능한 기능입니다. 새로운 모집공고를 작성할 수 있습니다. 모집공고 번호는 고유 번호이므로 겹칠 수 없습니다.

8) 모집공고 삭제
- 관리자만 사용가능한 기능입니다. 이전에 작성한 모집공고를 삭제할 수 있습니다.

3. 간단한 기능 사용 예시

1) 교환학생 프로그램 응시
관리자 로그인 -> 모집공고 작성
학생 로그인 -> 모집공고 조회 -> 모집공고 선택 -> 지원하기 선택

2) 이수학점 등록
학생 로그인 -> 이수학점 관리 -> 학기 선택 -> 이수학점 등록 -> 이수학점 관리에서 바로 확인 가능

3) QnA
학생 로그인 -> 질문게시판 -> 질문 작성
관리자 로그인 -> 질문게시판 -> 질문에 대한 답변 작성

4) 회원가입
로그인 화면 -> 회원가입 -> 아이디 입력 후 중복학인 -> 필요한 정보 입력 -> 회원 가입

5) 파견실적 조회
로그인 -> 파견실적 조회 -> 검색할 필터 선택(미선택도 가능) -> 조회
